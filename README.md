# 微信公众号文章批量抓取工具

批量抓取指定微信公众号的历史文章，保存为本地 HTML 文件，并支持按文章类型或部门自动分类。

## 项目结构

```
wx-article-spider/
├── scraper.py               # 抓取脚本（获取列表 + 下载文章）
├── classifier.py            # 分类脚本（按类型 / 按类型+部门）
├── requirements.txt         # Python 依赖
├── article_list.json        # [自动生成] 文章元数据列表（断点续传）
├── download_progress.json   # [自动生成] 下载进度（断点续传）
├── articles/                # [自动生成] 下载的文章 HTML 文件
├── articles_classified/     # [自动生成] 按类型分类后的文章
└── articles_by_dept/        # [自动生成] 按类型 + 部门分类后的文章
```

## 工作原理

微信公众平台（mp.weixin.qq.com）的文章编辑器支持插入任意公众号的文章链接，
其内部接口可以查询任意公众号的全部文章列表。

只需登录自己拥有的公众号的管理后台，即可获取任意公众号的所有文章，**无需拥有目标账号**。

## 环境准备

```bash
# 安装依赖
pip install -r requirements.txt

# 安装 Playwright 浏览器内核
playwright install chromium
```

> 需要拥有一个微信公众号账号用于登录微信公众平台。
> 没有的话可在 mp.weixin.qq.com 免费注册一个微信公众号。

## 使用方法

### 第一步：获取文章列表

```bash
# 指定要抓取的公众号名称
python scraper.py fetch --account "公众号名称"
```

运行后会自动打开浏览器，进入微信公众平台登录页：

1. 用微信扫描页面二维码
2. 在手机上点击「登录」确认
3. 登录成功后脚本自动继续，无需其他操作
4. 如果搜索结果有多个同名账号，脚本会列出供你选择

### 第二步：下载文章

```bash
# 下载全部文章
python scraper.py download

# 只下载最新 N 篇
python scraper.py download --count 200
```

### 第三步（可选）：分类整理

```bash
# 按文章类型分类（输出到 articles_classified/）
python classifier.py classify

# 按文章类型 + 部门分类（输出到 articles_by_dept/）
python classifier.py classify-dept
```

## 文章分类说明

`classifier.py` 使用基于关键词的规则引擎，将文章归入以下 25 个类别：

| 类别 | 说明 |
|------|------|
| 时政要闻 | 党政领导活动、重要讲话精神 |
| 党建思政 | 党建、思政教育、主题教育 |
| 悼念纪念 | 悼念、缅怀相关内容 |
| 安全提醒 | 反诈、安全警示通知 |
| 雄安校区 | 雄安相关动态 |
| 本科招生 | 高考招生、录取信息 |
| 研究生招生 | 考研、复试、研招信息 |
| 就业创业 | 就业指导、招聘信息 |
| 人才引进 | 教师及非教师岗位招聘 |
| 毕业季 | 毕业典礼、毕业相关活动 |
| 迎新季 | 新生报到、军训、开学典礼 |
| 竞赛获奖 | 各类竞赛获奖喜报 |
| 学术科研 | 学术论文、科研成果、学术会议 |
| 科技创新 | 科技成果、国家标准、创新项目 |
| 荣誉表彰 | 入选榜单、奖学金、荣誉称号 |
| 人物风采 | 师生个人风采故事 |
| 校庆校友 | 校庆活动、校友动态 |
| 国际交流 | 国际合作、海外交流 |
| 媒体聚焦 | 人民日报、央视等媒体报道 |
| 文化艺术 | 话剧、歌会、电影节等文艺活动 |
| 节日祝福 | 节日祝福、新年贺词 |
| 通知公告 | 放假安排、选课、校历等 |
| 校园活动 | 开放日、讲座、社团活动等 |
| 校园风光 | 花开、赏景、季节风光 |
| 校园生活 | 美食、打卡、心理健康等日常内容 |
| 校园发展 | 新学院、合作协议、校园建设 |
| 其他 | 未匹配到以上类别的文章 |

分类采用**标题优先**策略：优先匹配标题，标题无法判定时以正文辅助匹配兜底。

`classify-dept` 命令在类型分类基础上，还会从正文中提取「特别支持 | 部门名」署名，
输出结构为 `articles_by_dept/<部门>/<类型>/`，无署名文章归入 `未标注部门/`。

## 常见问题

### Q: 获取列表时提示"频率限制"

微信接口触发了频率控制，进度已自动保存。等待 10 分钟后重新运行，会从断点继续：

```bash
python scraper.py fetch --account "公众号名称"    # 重新扫码登录，从断点继续
python scraper.py download
```

### Q: 提示"session 过期"

token 失效，重新运行扫码即可，进度不会丢失：

```bash
python scraper.py fetch --account "公众号名称"
```

### Q: 搜索到多个同名账号

脚本会列出所有匹配结果，并显示账号简介，按提示输入编号选择即可：

```
找到以下匹配账号，请选择：
  1. 测试大学 - 测试大学官方公众号
  2. 测试大学研究生院 - ...
请输入编号 (1-2)：
```

### Q: 文章很多，想分批下载

先获取完整列表，再按需下载：

```bash
python scraper.py fetch --account "公众号名称"    # 获取全部列表（可能需要多次扫码）
python scraper.py download --count 200            # 先下载最新 200 篇
python scraper.py download                        # 之后再下载剩余的（自动跳过已下载）
```

### Q: 想重新下载全部

```bash
rm download_progress.json
rm -rf articles/
python scraper.py download
```

### Q: 想重新分类

分类命令每次运行会自动清空并重建输出目录，直接重新运行即可：

```bash
python classifier.py classify
python classifier.py classify-dept
```
